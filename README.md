# Usage 

1. Include style.css in your document

  `<link rel="stylesheet" href="style.css">`
  
2. Use any of the card style
	a. 
  [![Simple Card](https://github.com/PJijin/Simple-Card/blob/master/preview2.png?raw=true "Simple Card")]()
